CelOS 1

A simple, easy-to-use Linux distribution which uses Flatpak as the main package manager. CelOS proritises stability, ease-of-use and security to provide the best possible user-experience.
Created by The Cob:web-Aclevo Group

Why should you choose CelOS?

Here are some reasons as to why you should choose CelOS to be your Linux distribution of choice!


Flatpak at its core

Flatpak is a secure, easy-to-use package manager which allows you to download applications in a secure, sandboxed environment. We use Flatpak as the main package manager on CelOS due to its perfect integration with the GNOME Desktop and GNOME software, sandboxing and its ease-of-use.


Ease-of-use

The Linux Desktop typically has some form of learning curve. We attempt to make this the smoothest Linux experience for both new and experienced users. We have integrated Flatpak with GNOME software to provide a simple method of application installation and we come with the default GNOME desktop for a smooth user experience.


Security

All applications installed via Flatpak are sandboxed by default, meaning that you can choose its permissions on your system. Along with this we have Flatseal installed as a graphical way to manage permissions for applications installed via Flatpak.


Stability

CelOS aims to be a stable Linux distribution with a sturdy 2-year release-cycle. We are based on an LTS version of Ubuntu (Ubuntu 20.04 LTS) to ensure stability, however despite being an LTS, stable distribution we ensure that our applications remain modern and up-to-date with Flatpak!
